# CrisDesAnimaux
Avatar joue les son d'animaux

- les sons d'animaux: chien, blaireau, chauve_souris, fennec, herisson, chat, ane, cheval, elephant, agneau, bouc, chevre, mouton, chameau, vache, singe, canard, poule, poussin, ours, tigre, abeille, cigale.

- réponse aléatoire avec son d'un animal:  "Écoute bien", "Voici le son", "Tu reconnais cet animal", "Attention", "Devine l'animal".

- le régle est: fait/mets/joue Le cris de l'animal {chien ou chat, ect....}

- Version 1.0
